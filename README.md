**Bali Tech Hub** is the fastest growing technology and IT community in Bali, bringing together tech enthusiasts, developers, designers, entrepreneurs, and students to collaborate, learn, and innovate together. This repository contains the source code for the official website: [https://balitechhub.com](https://balitechhub.com).

## ğŸ’» Local Development Setup

1.  **Prerequisites:**
    - A local web server environment like [XAMPP](https://www.apachefriends.org/), [MAMP](https://www.mamp.info/), or PHP's built-in server.
    - Git

2.  **Clone the repository:**
    ```bash
    git clone https://github.com/dewatatech/balitechhub.git
    ```

3.  **Move to your web server directory:**
    - For XAMPP: Move the folder to `C:\xampp\htdocs\balitechhub`
    - For MAMP: Move to `/Applications/MAMP/htdocs/balitechhub`

4.  **Start the server:**
    - Start Apache via your control panel.
    - OR use PHP's built-in server from the project root:
      ```bash
      php -S localhost:8000
      ```

5.  **Access the site:**
    - Open your browser and navigate to `http://localhost/balitechhub` (or `http://localhost:8000`).
